# Responsive Navbar

A modern and responsive navigation menu built with **HTML, CSS and JavaScript**.

This project demonstrates how to create a clean and mobile-friendly navigation bar without using frameworks.

---

## 🚀 Features

- Responsive navigation menu
- Mobile hamburger menu
- Login and Sign Up buttons
- Clean and modern UI
- Beginner friendly project

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📂 Project Structure

responsive-navbar
│  
├── assets/css/style.css  
├── assets/js/script.js  
├── index.html
└── README.md  

---

## ▶️ How to Run

1. Clone the repository

git clone https://github.com/domingos-pk/responsive-navbar.git

2. Open the project folder

3. Open **index.html** in your browser.
